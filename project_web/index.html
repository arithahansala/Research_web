ABOUT OUR RESEARCH

Our research focuses on enhancing Quality of Service (QoS) in modern networks using Machine Learning-based traffic classification within a Software-Defined Networking (SDN) environment. Traditional networks struggle to manage increasing traffic complexity — especially with encrypted and dynamic data flows. To address this, we built an intelligent system that classifies network traffic in real time using deep learning models and applies adaptive QoS rules through the SDN controller. This ensures optimized bandwidth allocation, reduced latency, and smarter traffic prioritization. By integrating SDN programmability with the predictive power of machine learning, our work represents a significant step toward more efficient, scalable, and intelligent network management.

KEY FEATURES

SDN Based Netwrok

A network where a central software controller manages traffic flow and network devices dynamically for better control and flexibility

Traffic Classification

The process of identifying and categorizing network traffic based on its characteristics to manage and optimize data flow.

Adaptive QoS Enahncement

Automatically adjusts network Quality of Service settings to improve performance based on real-time traffic conditions.

Why this research matters more than ever

Enhancing QoS using machine learning-based traffic classification is crucial today due to the growing complexity and volume of internet traffic. Traditional methods struggle with encrypted and diverse traffic types, while ML can intelligently and dynamically identify traffic patterns, enabling networks to prioritize critical services, reduce latency, and improve user experience. Combined with SDN, ML allows real-time, automated, and adaptive QoS management, making networks more efficient, secure, and responsive to modern demands.

Mininet is a lightweight network emulator that creates a realistic virtual network using software. It allows you to simulate networks with hosts, switches, links, and controllers—all on a single machine. Widely used for testing and developing Software-Defined Networking (SDN) applications, Mininet provides a simple, flexible environment for rapid prototyping and experimentation.
